Detailed Description

P3_RGB_LED_matrix class.

This is a class that uses the hwlib library to provide the user with an interface to work with matrices like this one: https://www.adafruit.com/product/2279 This class provides a buffer with write functions and a toolset of other functions. The user must give three hwlib pins and three hwlib port_outs to the constructor for initialization.

draws pixels on the matrix

This function clocks in the data from the buffer to the shift-registers on the matrix. After clocking in one row on the matrix it will display the row for a short period before moving on to the next row. This function fully draws the screen one time.
